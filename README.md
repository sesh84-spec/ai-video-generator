# AI Video Generator (Telegram Bot + Web App)

Проект для генерации видео с помощью AI (Sora через Proxy API) с двумя интерфейсами:
- Telegram-бот
- Веб-интерфейс (Flask)

## 📌 Описание

Система позволяет пользователю:
- отправить текстовый промпт
- запустить генерацию видео
- отслеживать прогресс
- получить готовый MP4-файл

Проект построен по принципу:
- **backend** — работа с Proxy API (`request.py`)
- **frontend** — Telegram-бот и веб-интерфейс

---

## 🚀 Функционал

### Telegram-бот
- приём текстового запроса
- отображение прогресса генерации
- отправка готового видео прямо в чат

### Веб-интерфейс
- форма ввода промпта
- прогресс-бар
- получение `task_id`
- скачивание видео

🔄 Как работает генерация
создаётся задача (/videos)
происходит опрос статуса (queued → in_progress → completed)
после завершения скачивается MP4
файл отдаётся пользователю

⚠️ Особенности
генерация видео занимает время (очередь + обработка)
возможны ошибки модерации (moderation_blocked)
каждый запрос расходует баланс Proxy API
видео сохраняется локально

📌 Ограничения
длительность видео: 4 / 8 / 12 секунд
требуется платный API (бесплатных лимитов нет)
хранение задач реализовано в памяти (без БД)
