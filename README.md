# AI Video Generator

Локальный проект для генерации видео через Proxy API.

## Возможности
- генерация видео по текстовому промпту
- Telegram-бот
- веб-интерфейс на Flask
- отслеживание статуса генерации
- скачивание готового MP4

## Стек
- Python
- Flask
- pyTelegramBotAPI
- requests
- Proxy API
